# Welcome to Blink Code Editor

Blink Code Editor is a high-performance IDE built from the ground up to ensure every millisecond of latency is eliminated. This is a standalone development environment, not a VS Code fork, focusing on a lightweight footprint and instantaneous feedback.

## Why Blink?

Blink was designed with performance as the top priority. Every feature is carefully crafted to minimize latency and maximize responsiveness:

- **Zero bloat** - Only the essentials, nothing more
- **Instant startup** - Launch and start coding immediately
- **Native performance** - Built with Electron for cross-platform desktop power
- **Modern architecture** - React + TypeScript + Vite for maintainability
